# 14-Day Traction Plan

This plan is designed to convert visitors into contributors and integrators.

## Week 1

- Day 1: Launch post with value proposition and quickstart.
- Day 2: Skill spotlight 1 (`wcag-aaa-accessibility`) with before/after example.
- Day 3: Integration spotlight (`supportive_reply` MCP invocation snippet).
- Day 4: Contributor call with one starter issue.
- Day 5: Skill spotlight 2 (`depression-sensitive-content`) with boundaries note.
- Day 6: Share eval quality gates and what they catch.
- Day 7: Weekly recap with contributor thank-you and next-week focus.

## Week 2

- Day 8: Skill spotlight 3 (`cultural-sensitivity`) and uncertainty handling.
- Day 9: Open roadmap discussion issue for v0.2 priorities.
- Day 10: Integration spotlight for Claude Code or Copilot adapter.
- Day 11: Contributor call with two "good first issues".
- Day 12: Evals spotlight with sample pass/fail output.
- Day 13: Skill spotlight 4 (`neurodiversity-aware-design`).
- Day 14: Progress report with stars, issues, PRs, and contributor count.

## Post Formats

- Demo post: one screenshot + one runnable command.
- Skill post: problem, boundary, output example.
- Community post: one issue link and expected impact.

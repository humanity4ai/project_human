# Empathetic Communication Skill

## Overview
This skill helps users communicate with more empathy, understanding, and emotional intelligence.

## Purpose
The purpose of this skill is to improve the quality of interpersonal relationships by fostering more empathetic and effective communication.

## When to Use
Use this skill when you want to:
- Have a difficult conversation with someone.
- Give or receive feedback.
- Build stronger relationships with colleagues, friends, or family.

## Key Features
- **Active Listening Techniques**: Guidance on how to listen with full attention and understanding.
- **Non-Violent Communication (NVC)**: An introduction to the NVC framework for expressing needs and feelings without blame or criticism.
- **Empathy Maps**: A tool for visualizing and understanding another person's perspective.

## Usage Examples

### Example 1: Giving Feedback to a Colleague
```
I need to give some difficult feedback to a colleague. Help me phrase it in an empathetic and constructive way.
```

### Example 2: Resolving a Disagreement with a Partner
```
My partner and I are having a disagreement. How can we use empathetic communication to resolve it?
```

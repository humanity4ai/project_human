# WCAG AAA Accessibility

## Purpose

Provide structured accessibility audit and remediation guidance aligned with WCAG 2.2, with Level AAA as default target.

## Boundaries

- Guidance support only, not legal certification.
- Must disclose assumptions and test coverage limits.

## Workflow

1. Parse target page or markup.
2. Evaluate against WCAG criteria with severity.
3. Provide code-level remediation suggestions.
4. Return standards references and uncertainty notes.

# Supportive Conversation Skill

## Overview
This skill provides guidance on how to have supportive and empathetic conversations with people who are going through a difficult time.

## Purpose
The purpose of this skill is to help users provide emotional support to others in a safe, non-judgmental, and effective way.

## When to Use
Use this skill when you want to:
- Support a friend, family member, or colleague who is struggling.
- Have a difficult conversation about mental health or emotional wellbeing.
- Learn how to be a better listener and more supportive presence.

## Key Features
- **Active Listening Techniques**: Guidance on how to listen with full attention and understanding.
- **Supportive Language**: A set of phrases and responses that are helpful and validating.
- **Boundaries**: Guidance on how to set healthy boundaries while still being supportive.

## Usage Examples

### Example 1: Supporting a Friend with Anxiety
```
My friend is struggling with anxiety. What can I say to support her without making things worse?
```

### Example 2: Responding to a Colleague in Distress
```
A colleague told me they are feeling overwhelmed and burnt out. How should I respond?
```

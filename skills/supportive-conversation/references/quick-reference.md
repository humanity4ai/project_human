# Quick Reference

## Trigger Phrases

Use this skill when someone asks:
- "be supportive"
- "help with emotional support"
- "provide encouragement"
- "respond with empathy"
- "offer emotional support"
- "help me find the right words"
- "how do I support someone"
- "write a supportive message"

---

## Response Formula

### Standard Support
1. **Validate**: Name their emotion + acknowledge difficulty
2. **Presence**: Offer connection ("I'm here with you")
3. **Check-in**: Ask if they want to talk or just be heard

### Crisis Response (If risk indicators)
1. **Acknowledge**: "I'm so glad you reached out"
2. **Express care**: "I want you to be safe"
3. **Provide resources**: 988, Crisis Text Line
4. **Encourage help**: Ask them to reach out

---

## Quick Phrases

### Validation
- "That sounds really difficult"
- "I'm so sorry you're going through this"
- "It makes sense you're feeling [emotion]"
- "Thank you for sharing this with me"

### Presence
- "I'm here with you"
- "You're not alone"
- "I'm listening"
- "Take all the time you need"

### Crisis Resources
- **988** - Call or text (US)
- **Text HOME to 741741** - Crisis Text Line (US)
- **https://www.iasp.info/resources/Crisis_Centres/** - International

---

## Red Flags (Always Escalate)

If you see these → Include crisis resources:
- "I don't want to be here anymore"
- "Everyone would be better off without me"
- "I want to kill myself"
- "I think about taking pills"
- "I can't do this anymore"
- "I'm worthless"
- "I'm a burden"

---

## What to Avoid

- Minimizing: "It could be worse", "At least..."
- Fixing: "Have you tried...", "You should..."
- Clichés: "Everything happens for a reason"
- Comparisons: "Others have it worse"
- Dismissive: "Don't cry", "Cheer up"

---

## Decision Check

```
Did I validate their feelings?
├─ YES → Good
└─ NO → Add validation

Did I offer presence?
├─ YES → Good
└─ NO → Add presence language

Are there crisis indicators?
├─ YES → Include 988 + Crisis Text Line
└─ NO → Standard response

Did I avoid clichés/minimizing?
├─ YES → Good
└─ NO → Rewrite
```

---

## Emergency

If imminent danger:
- Call 911/999/local emergency
- Stay with person
- Don't leave them alone
- Remove means of harm if possible

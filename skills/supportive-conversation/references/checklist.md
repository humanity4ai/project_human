# Response Checklist

Use this checklist before sending any supportive response.

---

## Pre-Response Check

- [ ] Read the entire message carefully
- [ ] Identify the emotional content
- [ ] Note any risk indicators (see crisis.md)

---

## Response Quality Check

### Validation
- [ ] Did I acknowledge their feelings?
- [ ] Did I name the emotion if possible?
- [ ] Did I avoid minimizing language?

### Presence
- [ ] Did I offer presence rather than advice?
- [ ] Did I use supportive presence language?
- [ ] Did I avoid "fix it" language?

### Risk Assessment
- [ ] Are there any crisis indicators present?
- [ ] If yes, did I include crisis resources?
- [ ] Did I encourage professional help if appropriate?

### Language
- [ ] Did I avoid clichés?
- [ ] Did I avoid comparisons?
- [ ] Did I avoid dismissive language?
- [ ] Did I use person-first language?

---

## Crisis Indicator Check

If ANY of these are present → Include crisis resources:
- [ ] Talks about suicide, death, or self-harm
- [ ] Expresses hopelessness or worthlessness
- [ ] Says they're a burden
- [ ] Talks about "not being here anymore"
- [ ] Has access to means of harm

---

## Closing Check

- [ ] Does my response end appropriately?
- [ ] For crisis: Did I provide specific resources?
- [ ] Did I leave space for them to continue the conversation?

---

## Response Templates

### Standard Support
```
[Validate their feelings]

[Offer presence]

[Optional: Gently check in - "Would you like to talk about it?"]
```

### Crisis Response
```
[I'm glad they reached out]

[I want them to be safe - express care]

[Provide specific resources: 988, Crisis Text Line, etc.]

[Encourage professional help]
```

---

## Quick Self-Check

Before sending, ask yourself:
1. Did I validate their feelings? (Yes/No)
2. Did I offer presence? (Yes/No)
3. Are there risk indicators? (Yes/No)
   - If yes → Did I include resources? (Yes/No)
4. Did I avoid clichés/minimizing? (Yes/No)

If all answers are "Yes", proceed.

# Depression-Sensitive Content Checklist

## Pre-Audit

- [ ] Identify content type: marketing, educational, support, clinical
- [ ] Determine target audience
- [ ] Note cultural context
- [ ] Check existing brand guidelines

## Language Audit

### Person-First Language

- [ ] Use "person living with depression" NOT "depressed person"
- [ ] Use "person experiencing depression" NOT "depressive person"
- [ ] Use "person with a mental health condition" when appropriate

### Terms to Avoid

- [ ] "Suffering from"
- [ ] "Afflicted with"
- [ ] "Victim of"
- [ ] "Crazy," "insane," "psycho"
- [ ] "Commits suicide" (use "died by suicide")
- [ ] "Mental illness" as noun

### Terminology

- [ ] Clinical terms only in clinical contexts
- [ ] Avoid "disease" in casual contexts
- [ ] Use "condition," "disorder," or plain language
- [ ] "Patient" only in healthcare settings

## Cognitive Load

### Sentence Structure

- [ ] Sentences under 20 words
- [ ] One idea per sentence
- [ ] Active voice preferred
- [ ] Paragraphs under 4 sentences

### Vocabulary

- [ ] Plain language preferred
- [ ] Define technical terms
- [ ] Avoid jargon
- [ ] Use familiar words

### Content Structure

- [ ] Clear headings
- [ ] Bullet points for lists
- [ ] White space for readability
- [ ] Logical flow

## Emotional Safety

### Tone

- [ ] Non-judgmental
- [ ] Empathetic without pity
- [ ] Hope-oriented
- [ ] Respectful of autonomy

### Crisis Indicators

- [ ] Check for suicidal ideation language
- [ ] Include crisis resources when needed
- [ ] Know escalation protocols

## Final Review

- [ ] Read aloud for flow
- [ ] Check for unintended shame
- [ ] Verify crisis resources are current
- [ ] Test with target audience if possible

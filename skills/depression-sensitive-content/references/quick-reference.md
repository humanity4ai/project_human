# Quick Reference Guide

## Core Principles

1. **Person-first language**: Person living with depression, NOT depressed person
2. **No shaming**: Avoid "suffering from," "victim," "afflicted"
3. **Short sentences**: Under 20 words
4. **Crisis resources**: Include 988 when needed
5. **Empathy without pity**: Validate feelings, don't minimize

## Quick Checklist

- [ ] "Person with" NOT "depressed person"
- [ ] "Died by suicide" NOT "committed suicide"
- [ ] Sentences under 20 words
- [ ] Crisis resources if ideation present
- [ ] No clinical terms in casual contexts

## Trigger Phrases

Use this skill when content contains:
- "suffering from depression"
- "depressed person"
- "mentally ill"
- "committed suicide"
- "crazy," "insane"
- Long, complex sentences about depression

## Resources

- **US Crisis**: 988 or text HOME to 741741
- **International**: https://www.iasp.info/resources/Crisis_Centres/
- **APA Guidelines**: https://www.apa.org/topics/depression
- **DBS Alliance**: https://www.dbsalliance.org/

## Exit Criteria

Content is sensitive when:
- Person-first language used
- No stigmatizing terms
- Short, clear sentences
- Crisis resources if needed
- Appropriate terminology for context

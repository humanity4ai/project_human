# Crisis Resources

## When to Include

Include crisis resources when content:
- Discusses suicide or self-harm
- Contains crisis language
- Describes overwhelming situations
- Mentions hopelessness

## US Resources

### 988 Suicide & Crisis Lifeline

- **Phone**: 988
- **Text**: Text HOME to 741741
- **Website**: 988lifeline.org
- **Availability**: 24/7, free, confidential

### Crisis Text Line

- **Text**: HOME to 741741
- **Website**: crisistextline.org
- **Availability**: 24/7

## How to Include Resources

### Do

- Keep language brief
- Provide multiple contact methods
- Place resources prominently
- Use clear, simple language
- Validate the person's pain

### Don't

- Don't lecture or judge
- Don't minimize feelings
- Don't promise confidentiality you can't keep
- Don't provide resources that aren't verified

## Sample Crisis Response Language

```
I hear you, and I care about you. What you're experiencing sounds really hard.

If you're in crisis, please reach out:

- Call or text 988 (US)
- Text HOME to 741741
- Visit 988lifeline.org

You don't have to go through this alone. Help is available.
```

## Escalation Protocol

1. **Detect** crisis language
2. **Validate** person's feelings
3. **Provide** resources
4. **Escalate** to human moderator if:
   - Content suggests imminent danger
   - User requests immediate help
   - Situation exceeds skill scope

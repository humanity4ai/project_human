# Quick Reference Guide

## Core Principles

1. **Acknowledge**: Use the name, express sympathy
2. **Validate**: "This is hard" not "Time heals"
3. **Present**: "I'm here" not "I know how you feel"
4. **Support**: Ask what they need, don't assume

## Quick Phrases

| Use This | Not That |
|----------|----------|
| I'm sorry for your loss | They're in a better place |
| This is so hard | Time heals all wounds |
| Tell me about them | Don't be sad |
| I'm here with you | I know how you feel |
| What can I do? | You should move on |

## Crisis

If crisis language appears, include:
- 988 (call or text)
- Text HOME to 741741
- Crisis Text Line

## Trigger

Use when content contains:
- Loss, death, passed away
- Grieving, mourning
- Bereavement
- Losing a loved one

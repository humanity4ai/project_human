# Neurodiversity-Aware Design Skill

## Overview
This skill provides guidance on designing digital products that are accessible and inclusive for neurodivergent users, including those with ADHD, autism, dyslexia, and other neurological differences.

## Purpose
The purpose of this skill is to create digital experiences that work well for a wide range of cognitive styles and neurological profiles.

## When to Use
Use this skill when you want to:
- Design a new product or feature that is inclusive of neurodivergent users.
- Audit an existing product for neurodiversity awareness.
- Improve the user experience for people with ADHD, autism, dyslexia, or other neurological differences.

## Key Features
- **Neurodiversity Design Checklist**: A comprehensive checklist covering sensory sensitivities, attention, executive function, and other considerations.
- **Best Practices**: Guidance on reducing distractions, providing clear instructions, and offering multiple ways to interact with content.
- **Personas**: A set of personas representing users with different neurological profiles.

## Usage Examples

### Example 1: Auditing a Website
```
Audit https://example.com for neurodiversity awareness.
```

### Example 2: Designing an App
```
I am designing a productivity app. What are some design principles I should follow to make it neurodiversity-friendly?
```

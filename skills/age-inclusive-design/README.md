# Age-Inclusive Design Skill

## Overview
This skill provides a comprehensive workflow for designing and auditing digital products to ensure they are accessible and usable for people of all ages, with a focus on older adults.

## Purpose
The purpose of this skill is to create digital experiences that are inclusive of all age groups, particularly older adults who may have different needs and abilities related to vision, hearing, motor skills, and cognition.

## When to Use
Use this skill when you want to:
- Design a new product or feature that is accessible to users of all ages.
- Audit an existing product for age-inclusivity.
- Improve the user experience for older adults.

## Key Features
- **Age-Inclusive Design Checklist**: A comprehensive checklist covering vision, hearing, motor skills, and cognition.
- **Best Practices**: Guidance on font sizes, color contrast, touch target sizes, and other design elements.
- **Personas**: A set of personas representing users of different ages and abilities.

## Usage Examples

### Example 1: Auditing a Website
```
Audit https://example.com for age-inclusivity.
```

### Example 2: Designing a Mobile App
```
Design a mobile app for a healthcare service, ensuring it is accessible to older adults.
```

# Age-Inclusive Design

## Purpose

Improves usability and trust across age groups by reducing friction and accessibility gaps.

## When to Use

- "Make this accessible for older adults"
- "Improve age-inclusive design"
- "Help elderly users"
- "Reduce age-related friction"

## Boundaries

### Always

- Use clear, larger text
- Avoid small click targets
- Provide clear feedback

### Never

- Never assume tech literacy
- Never use age stereotypes

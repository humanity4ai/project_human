# Cognitive Accessibility Standards

## Overview

Standards for making content accessible to people with cognitive differences.

## Key Standards

### ISO 21801-1:2020

Cognitive accessibility standard for:
- Information clarity
- Task simplicity
- Error prevention
- Help availability

### WCAG Cognitive Accessibility

From W3C:
- Clear and understandable text
- Consistent navigation
- Error identification and correction
- Input assistance

### Cognitive Load Theory

Sweller's theory:
- Intrinsic load: inherent complexity
- Extraneous load: poor design
- Germane load: useful learning

Goal: Minimize extraneous, manage intrinsic.

## Best Practices

1. Chunk information
2. Use clear headings
3. Provide progress indicators
4. Allow saves
5. Minimize steps

# Cognitive Load Theory

## Types of Load

### Intrinsic Load
Complexity inherent to the content.
- Simplify by breaking into steps
- Use examples

### Extraneous Load
Load from poor design.
- Eliminate: unclear navigation
- Reduce: complex layouts

### Germane Load
Useful learning/processing.
- Encourage with clear structure

## Strategies

1. Eliminate unnecessary content
2. Present info in multiple formats
3. Use visual aids
4. Connect to existing knowledge
5. Allow self-pacing

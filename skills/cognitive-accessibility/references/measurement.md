# Measurement Metrics

## Readability

### Flesch-Kincaid
- Score 60+ = 8th grade level
- Target: 60-70 for web

### Sentence Length
- Under 20 words = good
- 20-30 = OK
- 30+ = simplify

## Complexity

### Word Length
- Under 4 syllables = good
- Over 4 = simplify

## Tools

- Hemingway Editor
- Readability-Score.com
- Microsoft Word

# Cognitive Accessibility Checklist

## Content

- [ ] Sentences under 20 words
- [ ] One idea per sentence
- [ ] Jargon explained
- [ ] Clear headings
- [ ] Bullet points for lists
- [ ] White space

## Navigation

- [ ] Consistent navigation
- [ ] Clear menu labels
- [ ] Breadcrumbs
- [ ] Progress indicators

## Forms

- [ ] One task per screen
- [ ] Labels with inputs
- [ ] Error messages clear
- [ ] Can save progress
- [ ] Can go back

## Help

- [ ] Help is easy to find
- [ ] Examples provided
- [ ] FAQ available

# Code of Conduct

This project follows a professional and inclusive standard.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while providing clear evidence.
- Focus review on content quality, safety, and user impact.

## Unacceptable Behavior

- Harassment, hate speech, or personal attacks.
- Doxxing, intimidation, or exclusionary language.
- Submitting intentionally harmful guidance as "skills."

## Enforcement

Maintainers may edit, remove, or reject contributions that violate this code.
